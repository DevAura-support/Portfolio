## 🚀 Getting Started Locally

If you want to spin up our portfolio on your own machine to see how we structured things, just follow these quick steps:

1.  **Clone the repository:**
```bash
    git clone [https://github.com/](https://github.com/)[your-username]/[your-repo-name].git
    ```
2.  **Navigate into the directory:**
```bash
    cd [your-repo-name]
    ```
3.  **Install the dependencies:**
```bash
    npm install
    # or yarn install / pnpm install
    ```
4.  **Run the development server:**
```bash
    npm run dev
    ```

Open `http://localhost:3000` (or the port specified in your terminal) in your browser to see it live.
